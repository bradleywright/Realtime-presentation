  9. The old way

    Polling (via JSONP and Ajax)

    • Long-polling - Ajax request that waits for server to close connection: 
      <?php
    // fake waiting for data
    sleep(rand(2,10));
    // send response
    header('Content-type: application/json');
    echo '{"data":"here","key":"value"}';
?>
    • Long-polling via JSONP - Long-lived request wrapped in a callback function: 
      callback({"data":"here","key":"value"});
